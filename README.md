# react-seller

这个项目是基于[黄轶](https://github.com/ustbhuangyi)老师的 [vue-sell](https://github.com/ustbhuangyi/vue-sell) 项目实现的， 样式上做了些改动  
我用react技术栈重写了这个SPA作为初步学习的成果

## 技术栈

- React
- React Router
- ES6
- SCSS
- better-scroll
- fetch api

## 安装

将项目拷贝到本地

```bash
git clone https://github.com/xinkule/react-sell.git
```

安装服务端依赖

```bash
cd react-sell
npm install
```

安装客户端依赖

```bash
cd client
npm install
```

## 运行

Windows用户可直接点击以下文件
```bash
run.bat
```

Mac用户需要同时开启服务端和客户端
```bash
npm start
cd client
npm start
```
